# ridge 设计系统规范

## 1. 设计理念

### 品牌人格
**登山向导** — 温暖可靠、与你并肩、带你看得更远

向导不抢戏，只在岔路口举旗。温暖但不失专业，像向导的手套，粗糙但可靠。

### 装备语言
**碳纤维 + 钛合金** — 现代专业、轻量、精密

温暖的高科技向导：像一位使用现代装备的资深向导，既专业可靠，又不冷漠。

### 核心张力
- 碳纤维/钛合金 = 冷峻、科技、精密
- 山脊向导 = 温暖、自然、人文

**解决方案**：50/50 对等融合 — 温暖米白背景 + 钛合金精密浅灰 + 珊瑚金路标点缀

## 2. 色彩系统

### 2.1 色彩结构

```
背景层（山脊晨光）#faf9f6
    ↓
卡片层（钛合金高光）#ffffff
    ↓
次级层（钛合金磨砂）#f5f2ed
    ↓
强调色（向导路标）#e07a5f
```

### 2.2 色彩原则

| 层级 | 名称 | 用途 |
|------|------|------|
| 背景 | 山脊晨光 | 页面背景，温暖米白 |
| 卡片 | 钛合金高光 | 主卡片表面，纯白 |
| 次级 | 钛合金磨砂 | 次级卡片、输入框，浅沙色 |
| 分割线 | 钛合金阴影 | 分割线、边框，淡米色 |
| 强调 | 向导路标 | Primary按钮、选中态，珊瑚金 |

### 2.3 强调色使用规则（路标式）

- ✅ **仅关键决策点**：Primary按钮、选中态、当前导航、重要提示
- ❌ **不使用于**：大面积背景、装饰性图标、分割线、次要按钮

## 3. 形状系统

### 3.1 圆角定义

| 尺寸 | 值 | 用途 |
|------|-----|------|
| 微小 | `4px` | 标签、徽章（精密） |
| 标准 | `6px` | 按钮、输入框（machined edge） |
| 友好 | `8px` | 卡片、容器（标准） |
| 开阔 | `10px` | 大卡片、弹窗（视野开阔） |

### 3.2 形状语言

**精密圆润（6-8px）** — 钛合金的 machined edge

- 不锋利（不像精密仪器的2px）
- 不圆润（不像有机形状的16px）
- 刚刚好：精确机加工后的友好握持感

## 4. 交互系统

### 4.1 隐喻

**微光闪烁** — 像阳光在钛合金表面的反射变化

### 4.2 状态定义

| 状态 | 表现 | 时长 |
|------|------|------|
| Hover | 微妙的光泽亮度提升 +5% | 150ms |
| Active | 亮度提升 +10%，轻微内陷感 | 100ms |
| Focus | 柔和的光晕（ring），像阳光透过薄雾 | 200ms |
| Disabled | 灰度化 + 透明度50%，像被云雾遮蔽 | 即时 |

### 4.3 动效曲线

```css
transition: all 150ms cubic-bezier(0.4, 0, 0.2, 1);
```

## 5. 使用规则

### 5.1 层次规则

```
Page（背景）
  └── Card（卡片，圆角8px，无边框）
        └── Content
              └── Button（Primary，圆角6px）
              └── Input（圆角6px）
```

### 5.2 边框规则

- **卡片边框**：无边框 — 卡片通过背景色差异和阴影区分层次
- **输入框边框**：Focus时变为珊瑚金
- **分割线**：极淡暖色，或透明

### 5.3 阴影规则

**扁平但有层次** — 像山的高度差

| 层级 | 阴影 | 用途 |
|------|------|------|
| 无 | `none` | 默认状态，扁平 |
| 轻微 | 大地棕色调 4% | Hover提升感 |
| 中等 | 大地棕色调 8% | 下拉菜单、弹窗 |
| 强烈 | 大地棕色调 12% | 模态框、重要提示 |

## 6. 技术实现

### 6.1 CSS变量

完整 CSS 变量定义见：
- **主题文件**：`packages/web/src/assets/ridge.css`

### 6.2 组件库

**必须使用 shadcn-vue 组件**，禁止手写组件：
- Button、Card、Badge、Avatar、Input、Select
- Sidebar（导航）、Separator（分割线）、ScrollArea（滚动区）

样式完全通过 CSS 变量控制，禁止覆盖组件样式。

### 6.3 主题注册

- **注册文件**：`packages/web/src/assets/registry.ts`
- **默认主题**：`ridge`

## 7. 验收标准

- [ ] 卡片无边框，通过背景色和阴影区分层次
- [ ] 卡片有微妙的边界感，但不抢戏
- [ ] 珊瑚金只在关键决策点使用，不滥用
- [ ] 圆角在 4-10px 范围内，体现精密圆润
- [ ] 交互反馈像阳光反射，不是机械开关
- [ ] **全部使用 shadcn-vue 组件，无手写元素**
- [ ] 整体感觉是温暖的高科技向导
