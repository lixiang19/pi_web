# ridge 设计系统

## 核心原则

**框架与皮肤分离** — 设计系统定义永恒的布局与交互规则，主题系统提供可切换的配色方案。

---

## 永恒规则（不随主题变化）

### 间距系统

基于 4px 网格：

| Token | 值 | 用途 |
|-------|-----|------|
| `space-1` | 4px | 紧凑间隙 |
| `space-2` | 8px | 按钮内边距 |
| `space-3` | 12px | 卡片内边距 |
| `space-4` | 16px | 区块间距 |
| `space-5` | 24px | 大区块分隔 |
| `space-6` | 32px | 模块间距 |

### 圆角系统

| Token | 值 | 用途 |
|-------|-----|------|
| `radius-sm` | 4px | 标签、徽章 |
| `radius-md` | 6px | 按钮、输入框 |
| `radius-lg` | 8px | 卡片、面板 |
| `radius-xl` | 12px | 大卡片、模态框 |

### 字体系统

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
monospace: "SF Mono", Monaco, "Fira Code", monospace;
```

| 级别 | 大小 | 行高 | 字重 |
|------|------|------|------|
| `text-xs` | 12px | 16px | 400 |
| `text-sm` | 13px | 18px | 400 |
| `text-base` | 14px | 20px | 400 |
| `text-md` | 15px | 22px | 400 |
| `text-lg` | 16px | 24px | 500 |
| `text-xl` | 18px | 26px | 600 |
| `text-2xl` | 20px | 28px | 600 |

### 动效系统

| 类型 | 时长 | 曲线 |
|------|------|------|
| 颜色/透明度 | 150ms | ease |
| 尺寸/位置 | 200ms | ease-out |
| 列表进入 | 100ms | ease-out |

### 阴影系统

| Token | 值 |
|-------|-----|
| `shadow-sm` | `0 1px 2px rgba(0,0,0,0.1)` |
| `shadow-md` | `0 4px 12px rgba(0,0,0,0.15)` |
| `shadow-lg` | `0 8px 24px rgba(0,0,0,0.2)` |

# 主题系统

设计系统的皮肤层。每个主题提供完整的明暗两套配色。

## 主题 Token 规范

每个主题必须定义以下 token：

### 背景层
- `bg-base` — 最底层背景
- `bg-surface` — 卡片、面板背景
- `bg-elevated` — 悬浮层、下拉菜单

### 边框层
- `border-subtle` — 分隔线
- `border-default` — 组件边框

### 文字层
- `text-primary` — 主要文字
- `text-secondary` — 次要文字
- `text-tertiary` — 占位符、禁用

### 强调层
- `accent` — 主强调色
- `accent-hover` — 悬停态

### 状态层
- `success` — 成功
- `warning` — 警告
- `error` — 错误

## 主题扩展

新增主题只需提供上述 token 的明暗两套值。主题文件保持独立，运行时动态加载。

---

## 组件原型（使用主题 Token）

### 按钮

**主要**
```
背景: accent
文字: white（或根据 accent 亮度自动黑白）
圆角: 6px
内边距: 8px 12px
```

**次要**
```
背景: bg-surface
边框: 1px border-default
文字: text-primary
```

### 输入框
```
背景: bg-surface
边框: 1px border-default
聚焦: 边框 accent
占位符: text-tertiary
```

### 列表项
```
悬停: 背景 bg-elevated
选中: 背景 accent/10, 文字 accent
```
