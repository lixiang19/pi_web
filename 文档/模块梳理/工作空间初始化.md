# 工作空间初始化

## 职责

服务器启动时确保唯一 ridge 工作空间存在，并补齐预制目录和核心入口文件。

## 入口

- `packages/server/src/workspace-chat.ts`
- `ensureWorkspaceTemplate(workspaceDir)`

## 初始化内容

- 可见目录：`项目/`、`笔记/`、`日记/`、`剪藏/`、`附件/`、`记忆/`、`Wiki/`、`空间/`
- 隐藏目录：`.ridge/`
- 初始文件：`记忆/MEMORY.md`、`Wiki/index.md`

## 边界

- 重复启动只补缺失内容，不覆盖已有 Markdown 文件。
- 工作空间根路径存在但不是目录时失败。
- 预制目录路径被文件占用时失败。
- 初始化不创建外部项目、不创建 RAG/Kuzu 缓存实现、不迁移旧数据。
