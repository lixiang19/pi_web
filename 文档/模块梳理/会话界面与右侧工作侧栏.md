# 会话界面与右侧工作侧栏

## 职责

- 会话主界面（WorkbenchChatPanel + WorkbenchMessageStream + WorkbenchComposer）负责消息流展示、用户输入、复制消息、编辑旧用户消息、重试 AI 回复。
- 右侧工作侧栏（SessionWorkspaceSidebar）展示摘要、文件树、Git、Diff 四个标签。
- 编辑旧用户消息和重试 AI 回复在普通会话中创建分叉新会话；任务处理会话禁用此操作。
- 工作空间/内部项目使用隐藏版本管理 Diff；外部 Git 项目使用项目 Git Diff。

## 关键文件

- `packages/web/src/components/workspace/SessionWorkspaceSidebar.vue` — 右侧侧栏，含 summary/filetree/git/diff 标签。
- `packages/web/src/components/workspace/WorkspaceChatTab.vue` — 会话容器，串联 ChatPanel 和 Sidebar。
- `packages/web/src/components/workbench/chat/WorkbenchChatPanel.vue` — 聊天主面板（消息流 + composer）。
- `packages/web/src/components/workbench/chat/WorkbenchMessageStream.vue` — 消息流渲染（展开/折叠轮次）。
- `packages/web/src/components/chat/ChatMessageItem.vue` — 单条消息渲染（含复制按钮）。
- `packages/web/src/composables/usePerSessionChat.ts` — 每会话聊天状态，含 `forkFromUserMessage`、`forkFromAssistantRetry`。
- `packages/server/src/index.ts` — 创建会话时禁止从 task session 分叉（403）。
- `packages/server/src/session-payload.ts` — `buildSessionSummaryFromIndex` 提供会话元数据给 sidebar。

## 接口契约

- `SessionSummary` 包含 `sessionType`、`contextType`、`taskId`、`deviceId`、`runLocation`，用于 sidebar 判断项目类型和运行位置。
- `GET /api/sessions` — 返回会话列表（含上述元数据字段）。
- `POST /api/sessions` — `parentSessionId` 指向 task session 时返回 403。
- Sidebar props: `sessionId`, `sessionTitle`, `fileTreeRoot`, `projectType`, `isGit`, `isOnline`, `messageCount`, `roundCount`。
- `projectType` 规则：`ridge:workspace-chat` 项目为 `internal`，其他为 `external`。
- Diff 标签规则：`internal` → hidden placeholder；`external` + `isGit` → git diff placeholder；`external` + `!isGit` → unavailable。
