# 任务系统项目绑定与筛选

## 职责

任务（`workspace_tasks`）和里程碑（`workspace_milestones`）可绑定到项目（`projects`），支持项目维度筛选。

## 入口

- `packages/server/src/task-system.ts`
- `packages/server/src/routes/workspace-tasks.ts`
- `packages/web/src/composables/useWorkspaceTasks.ts`
- `packages/web/src/components/workspace/TaskView.vue`

## 数据模型

### `workspace_milestones`

- 新增可空列：`project_id TEXT`（通过 migration 版本 7 和 repair columns 自动补齐）

### `workspace_tasks`

- 新增可空列：`project_id TEXT`（同上）
- 创建任务时若未显式传 `projectId`，但所选里程碑有 `projectId`，则自动继承。

## API 变更

### 创建里程碑

```
POST /api/workspace/milestones
{
  title, goal, acceptanceCriteria, dueDate?, color?, projectId?: string | null
}
```

### 更新里程碑

```
PATCH /api/workspace/milestones/:id
{
  projectId?: string | null,
  ...
}
```

### 创建任务

```
POST /api/workspace/tasks
{
  title, priority, acceptanceCriteria, dueDate?, milestoneId?, projectId?: string | null
}
```

- 若未传 `projectId` 且里程碑存在 `projectId`，自动继承。

### 更新任务

```
PATCH /api/workspace/tasks/:id
{
  projectId?: string | null,
  ...
}
```

### 列表筛选

```
GET /api/workspace/tasks?projectId=none   // 仅返回无项目的任务
GET /api/workspace/tasks?projectId=<id>     // 仅返回指定项目的任务
GET /api/workspace/tasks                  // 返回全部
```

## 前端变更

- `useWorkspaceTasks` 新增 `projectFilter: Ref<string | null | undefined>`，加载时透传给 `getWorkspaceTasks(filter)`。
- `TaskView.vue` 新增项目筛选控件（全部 / 无项目 / 具体项目）。
- 创建/编辑任务和里程碑时可选择项目或无项目。

## 边界

- `projectId` 为可空列，不影响现有无项目数据。
- 筛选参数前后端一致：`none` 字符串表示“无项目”。
- 保持现有看板/列表/日历/里程碑行为不退化。

## 测试覆盖

## 模块关系

```
TaskView.vue
  → useWorkspaceTasks (projectFilter)
    → getWorkspaceTasks(projectFilter) / createWorkspaceTask(projectId)

```
