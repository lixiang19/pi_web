# 工作空间主页与统一输入框

## 主页定位

工作空间主页是全局 AI 输入入口，不是固定会话。

- 用户输入前显示输入框和轻量摘要。
- 用户提交后，主页内直接变成新工作空间会话。
- 新会话立即加入左侧“工作空间会话”列表并保持选中。
- 会话标题先用用户首句临时展示，之后由 AI 自动生成。

## 主页摘要

主页轻量摘要包含：

- 待处理闪念；
- 待处理任务；
- 审核中任务；
- 最近工作空间文件。

摘要自动实时刷新。

点击闪念摘要打开“闪念”页。

最近文件只来自 `~/ridge-workspace`，不包含外部项目文件。

## 主页布局

- 输入框位于页面中下方。
- 输入框上方显示三个快捷动作：处理闪念、规划任务、总结最近文件。
- 点击快捷动作只填入输入框，由用户确认后发送。

## 统一输入框

所有 AI 会话入口使用同一套输入框。

适用入口：

- 工作空间主页；
- 工作空间会话；
- 项目新会话；
- 任务处理会话。

第一版能力：

- 文本输入；
- 会话附件上传；
- 从右侧文件列表拖入文件引用；
- 模型选择；
- 思路强度选择；
- Agent 选择；
- 发送与中断。
- 发送错误可见且保留草稿；
- 归档会话只读时禁用输入和发送。

## 附件与文件引用

- 上传附件只属于当前会话上下文，不自动进入知识库或闪念。
- 从右侧文件列表拖入输入框时，只记录文件引用。
- 文件引用发送时由当前运行位置读取最新内容。

## 模型、思路强度、Agent

输入框底栏常显模型、思路强度、Agent 三个选择器。

- 新会话默认模型来自全局设置。
- 新会话默认思路强度来自全局设置。
- 新会话默认 Agent 来自全局默认 Agent。
- 会话创建后允许切换模型、思路强度和 Agent。
- 切换只影响后续消息，不写回全局默认。
- Agent 切换不需要在会话中额外记录。
- 任务处理会话强制使用任务 Agent。

## 会话 API 主线

- 首发创建：`POST /api/sessions`。
- 后续发送：`POST /api/sessions/:id/messages`。
- 历史读取：`GET /api/sessions/:id/messages`。
- 流式事件：`GET /api/sessions/:id/events`。
- 中断运行：`POST /api/sessions/:id/cancel`。
- Ask 回答：`POST /api/sessions/:id/ask/:requestId`。
- Permission 审批：`POST /api/sessions/:id/permissions/:requestId`。
