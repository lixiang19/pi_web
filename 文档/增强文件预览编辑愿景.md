# 增强文件预览编辑愿景

## 一句话定义

ridge 的文件树不只是浏览工具，而是**支持多格式预览、轻量编辑、与对话深度关联的迷你 IDE**。

---

## 核心能力

### 预览层
| 格式 | 预览方式 |
|------|---------|
| 图片 | 内置图览，支持缩放、切换 |
| Markdown | 渲染预览 + 源码切换 |
| 代码 | 语法高亮，支持语言切换 |
| PDF | 基础浏览（可选） |
| 结构化数据 | JSON/YAML/CSV 格式化查看 |

### 编辑层
- **轻量编辑**：点击文件直接内联修改，保存后自动触发会话刷新
- **草稿模式**：未保存改动标红点，支持放弃/确认
- **多标签**：类似 VS Code 的标签栏，切换已打开文件
- **Diff 视图**：文件修改前后对比，对话中展示变更

### 对话关联
- **@文件提及**：输入框 @ 选择文件，自动附带上下文到 prompt
- **文件变更卡片**：代码编辑后，对话流中显示"文件已修改"卡片
- **一键应用**：Assistant 建议的代码块，一键写入指定文件

```

│ 会话列表 │   对话区       │   文件区      │
│         │               │ ┌───────────┐ │

│         │               │ │ (目录浏览) │ │

│         │               │ │ 预览/编辑区 │ │
│         │               │ │ - 图片查看  │ │

│         │               │ │ - Markdown │ │
│         │               │ └───────────┘ │
│         │               │ ┌───────────┐ │

│         │               │ │ (多文件)   │ │
│         │               │ └───────────┘ │
└─────────┴───────────────┴───────────────┘
```

## 移动端适配

| 功能 | 移动端策略 |
|------|-----------|
| 文件树 | 隐藏，仅通过 @提及 选择文件 |
| 预览 | 支持图片、Markdown 查看 |
| 编辑 | ❌ 禁用，提示"请在桌面编辑" |

---

## 演进阶段

- **当前**：基础文件树，点击复制路径到输入框
- **一阶段**：图片/Markdown 预览，单文件只读
- **二阶段**：代码编辑 + 保存，标签页多文件
- **三阶段**：Diff 视图，一键应用 assistant 建议
