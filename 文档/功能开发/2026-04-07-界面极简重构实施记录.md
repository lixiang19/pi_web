# 界面极简重构实施记录 (2026-04-07)

## 重构目标
将原有的“带有厚重圆角、发光、暗色调”的 UI 彻底重构为“极简、现代、工业级”的设计语言。

## 核心变更

### 1. 结构与布局 (Platform Shell)
- **去圆角化**：移除了所有的 `rounded-xl` / `2xl`，改为直角边缘（Square Edges）。
- **极简侧边栏**：实现可折叠侧栏，移除顶部不必要的描述信息层。
- **空间利用**：使用 `divide-x` 和 `divide-y` 代替 Gap 间距，形成严谨的网格感。

### 2. 视觉语言 (Visual Language)
- **瑞士平面设计风格 (Swiss Design)**：
    - 大量使用 **全大写 (Uppercase)**、**极重字重 (Black/ExtraBold)**、**微小字号 (9px-11px)**。
    - 强调字间距 (Tracking-widest) 与排版对比。
- **激活态指示**：放弃色块包裹，改为 **左侧垂直边框 (Border-l-2)** 作为主要指示器。
- **主题兼容**：重构了 `useThemePreferences` 的应用逻辑，确保明亮模式（Light Mode）能完整覆盖所有背景色。

### 3. 组件重设计
- **SessionSidebar**：移除“卡片”感，改为纯净的列表流，搜索框改为硬核边框样式。
- **WorkspaceFileTree**：移除毛玻璃和独立容器，直接融入面板背景。
- **WorkbenchPage**：从 `gap-4` 的独立卡片布局转变为 `divide-x` 的三栏一体化编辑器布局。

### 4. 界面实验室 (Interface Design Lab)
新增了 `/design` 路由，作为设计实验场：
- **三种侧边栏风格对比**：
    - `BLOCK_MINIMAL`: 现代色块感。
    - `LINE_GUIDED`: 线条引导感。
    - `MONO_TERMINAL`: 怀旧终端感的单色设计。
- **概念性组件**：
    - `SWISS_GRID_TABLE`: 极致的网格化表格。
    - `MODERN_NOTIFICATION`: 基于垂直线条的极简提示。
    - `MINIMAL_ACTIONS`: 硬边框全大写按钮组。
