# 待办系统细化

> 状态：✅ 已修复
> 优先级：P1

## 问题

1. **checkbox 任务状态切换无法回写** — 只更新独立任务 DB，checkbox 来源的行切换无效
2. **清单视图无删除按钮** — 看板有删除但清单没有
3. **checkbox 扫描全量递归无缓存** — 每次请求扫描所有 .md 文件

## 修复方案

### 1. checkbox 回写 API

后端新增 `PATCH /api/workspace/tasks/checkbox`：

```json
// 请求
{
  "path": "项目/工作空间设计.md",
  "lineNumber": 15,
  "done": true
}
```

后端读取该文件，找到第 lineNumber 行，替换 `- [ ]` ↔ `- [x]`，写回文件。

前端 `toggleStatus` 对 `source.type === "checkbox"` 的行调此 API，而非 `updateWorkspaceTask`。

### 2. 清单视图添加删除按钮

为独立任务行添加 hover 显示的删除按钮（与看板一致）。checkbox 来源的行不显示删除（只能去原文件删除）。

### 3. checkbox 扫描缓存（暂不做）

全量扫描的性能问题在文件数 < 200 时不明显，标记为后续优化。

## 验收标准

- [x] checkbox 来源的待办点击切换后 .md 文件同步更新
- [x] 刷新页面后 checkbox 状态与 .md 文件一致
- [x] 清单视图独立任务有删除按钮
- [x] checkbox 来源的行无删除按钮（只有原文件可删）
