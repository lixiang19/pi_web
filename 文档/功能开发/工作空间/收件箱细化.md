# 收件箱细化

> 状态：✅ 已修复 + 全面升级
> 优先级：P1

## 原始问题

1. **收件箱没有历史列表** — 只有输入框，写完闪念就看不到了
2. **说明文案提到"拖拽归档"但拖拽不存在** — 误导用户

## 升级改进

### P0 完成

| 改进 | 说明 |
|------|------|
| 归档功能 | 右键/hover → 归档到...，InboxArchiveDialog 选择目标目录，调用 `moveFileEntry` API |
| 删除 + 确认 | InboxDeleteDialog 确认对话框，符合 Tauri 禁用原生 confirm 规范 |
| 内容预览 | 列表项显示首行内容预览（跳过 frontmatter/标题），最多20条预加载 |
| 空态重设计 | 图标+引导文案"写下你的第一个闪念"，对标 TaskView 空态 |
| Toast 反馈 | 捕捉/归档/删除/重命名 成功/失败均有 toast 提示 |
| 相对时间 | "3分钟前" 替代绝对日期，更符合闪念场景 |
