# 35 task review agent 任务回顾

## 目标

实现 task review agent，定期或由用户触发回顾任务和里程碑。

## 范围

- 扫描任务。
- 扫描里程碑。
- 读取处理会话索引。
- 读取最近 daily。
- 发现过期、阻塞、可拆分、可推进、需确认完成、里程碑风险和不一致。
- 输出建议到通知中心和任务页。

## 不做

- 不自动完成任务。
- 不自动执行任务。
- 不直接批量修改任务。

## 验收

- 建议必须用户确认后才写入正式任务或里程碑。
- 可建议标记阻塞、拆分、调整优先级、推进下一步、确认完成。
- 与其他后台 Agent 不并行修改同一任务。

## Spec 提取点

- task review 输入。
- 建议 schema。
- 接受建议动作。

## Spec 草案

### 输入

- 任务列表。
- 里程碑列表。
- 处理会话索引。
- 最近 daily。

### 输出

- suggestion_type
- related_task_id
- title
- reason
- proposed_action

### 行为

- 只生成建议。
- 建议进入通知中心。
- 用户接受后才调用任务 API。

### 测试

- 过期任务产生提醒建议。
- blocked 太久产生推进建议。
- 建议不会直接修改任务。
- 接受建议后任务变更。
