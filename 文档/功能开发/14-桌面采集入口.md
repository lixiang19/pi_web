# 14 桌面采集入口

## 目标

实现桌面端菜单栏采集入口。

## 范围

- 桌面端必须登录服务器。
- 菜单栏常驻入口。
- 文字采集。
- 区域截图、窗口截图、全屏截图。
- 文件采集。
- 剪贴板采集。
- 当前选区采集。
- 浏览器网址采集。
- 录音采集。

## 不做

- 采集时不选择项目。
- 录音采集不立即转写。
- 采集内容不直接进 RAG。

## 验收

- 采集结果统一进入服务器闪念。
- 浏览器网址权限不足时明确失败。
- 截图或文件允许没有文字说明。
- 桌面离线不可用。

## Spec 提取点

- 桌面到服务器上传协议。
- 各采集类型 payload。
- 权限失败处理。

## Spec 草案

### 采集类型

- text
- screenshot_region
- screenshot_window
- screenshot_fullscreen
- file
- clipboard
- selection
- browser_url
- audio

### 行为

- 桌面端必须登录服务器。
- 采集 payload 上传到服务器并创建闪念。
- 文件和截图作为闪念临时附件。
- 浏览器网址权限不足时返回明确错误。

### 测试

- 文本采集创建 text 闪念。
- 截图采集创建附件闪念。
- 未登录桌面采集失败。
- 浏览器权限失败不创建半成品闪念。
