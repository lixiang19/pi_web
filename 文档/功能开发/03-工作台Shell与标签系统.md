# 03 工作台 Shell 与标签系统

## 目标

实现 ridge 主工作台壳层和右侧标签系统。

## 范围

- 平台 Shell。
- 左侧导航区域。
- 右侧标签栏。
- 标签打开、激活、关闭。
- 单例标签和多例标签规则。
- 主页标签。
- 会话标签。

## 不做

- 不实现各功能页完整业务。
- 不做移动端完整设计。
- 不恢复旧独立页面路由作为入口。

## 验收

- 固定入口只操作工作台标签。
- 单例标签重复点击只激活。
- 终端允许多标签。
- 空主页标签关闭不产生会话。

## 关联设计

- `文档/项目设计/左侧导航与项目会话列表.md`
- `文档/项目设计/工作空间主页与统一输入框.md`

## Spec 提取点

- 标签类型。
- 标签生命周期。
- 单例和多例规则。
- 路由与标签状态关系。

## Spec 草案

### 标签类型

- home
- conversation
- singleton_feature
- terminal
- space_preview

### 行为

- 固定入口打开或激活标签，不跳旧页面路由。
- 单例功能标签重复打开只激活。
- 终端标签允许多开。
- 空主页关闭不创建会话。
- 会话标签关闭不删除会话。

### 测试

- 重复点击任务入口只保留一个任务标签。
- 连续点击终端入口生成多个终端标签。
- 关闭空主页后无 session_index 新记录。
- 打开同一会话只激活已有标签。
