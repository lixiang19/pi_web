# 24 全局搜索资产导航器

## 目标

实现左侧固定入口的全局搜索页面。

## 范围

- 搜索单例标签。
- 搜索文件、任务、里程碑、项目、会话索引、记忆、Wiki、空间、RAG、图谱节点。
- 类型分组。
- 类型、时间、项目、目录筛选。
- 结果点击打开对应标签。
- 搜索索引状态展示。

## 不做

- 不启动 AI。
- 不生成回答。
- 不搜索外部项目文件。
- 不搜索桌面本机项目文件和 Pi 会话正文。

## 验收

- 搜索结果确定、可浏览、可筛选。
- RAG 结果能打开来源文件。
- 图谱节点能打开邻接视图。
- 空间结果打开空间预览标签。

## 关联设计

- `文档/项目设计/全局搜索.md`
- `文档/项目设计/左侧导航与项目会话列表.md`

## Spec 提取点

- 搜索 API。
- 聚合排序。
- 结果类型 schema。

## Spec 草案

### API

- `GET /api/workspace/search?q=&type=&project=&time=&dir=`

### 结果类型

- file
- task
- milestone
- project
- session
- memory
- wiki
- space
- rag
- graph

### 行为

- 多路查询后聚合排序。
- 不启动模型。
- 点击结果打开对应标签。

### 测试

- 搜索任务标题返回 task 结果。
- 搜索 Markdown 内容返回 file/RAG 结果。
- 外部项目文件不返回。
- 结果可按类型筛选。
