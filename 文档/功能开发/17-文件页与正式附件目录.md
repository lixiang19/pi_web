# 17 文件页与正式附件目录

## 目标

实现工作空间文件页和正式附件目录展示。

## 范围

- 文件树展示工作空间可见目录。
- 隐藏 `.ridge`。
- 正式附件目录 `附件/`。
- 文件打开。
- Markdown 文件查看。
- 文件处理状态展示。
- 正式附件可被引用。

## 不做

- 不展示临时附件。
- 不做复杂文件编辑器。
- 不读取外部项目文件。

## 验收

- `附件/` 是用户可见长期资产。
- 正式附件可进入 RAG。
- `.ridge` 不作为普通目录展示。
- 文件页以服务器工作空间为边界。

## 关联设计

- `文档/项目设计/工作空间文件组织.md`
- `文档/项目设计/文件处理流程.md`

## Spec 提取点

- 文件树 API。
- 可见/隐藏判定。
- 文件状态字段。

## Spec 草案

### API

- `GET /api/workspace/files/tree`
- `GET /api/workspace/files/read`

### 行为

- 仅返回工作空间可见文件。
- 隐藏 `.ridge` 和临时目录。
- 正式附件目录可见。
- 文件节点显示处理状态。

### 测试

- 文件树包含八个预制目录。
- 文件树不包含 `.ridge`。
- 正式附件可读取。
- 临时附件读取失败。
