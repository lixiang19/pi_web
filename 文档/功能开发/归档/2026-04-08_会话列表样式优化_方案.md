# 会话列表样式优化方案
## 1. 背景与问题
- 当前会话列表使用大量 `text-muted-foreground/40` 等低透明度，导致看不清
- 搜索筛选功能不需要，增加视觉噪音
- 项目/分组嵌套层次过深，显得杂乱
## 2. 目标与非目标
- 目标：提升可读性、删除筛选、简化布局、保持明暗主题兼容
- 非目标：改变数据结构、添加新功能
## 3. 现状探索
- 关键文件：
  - [SessionSidebar.vue](packages/web/src/components/chat/SessionSidebar.vue) - 主容器，含搜索框
  - [SessionSidebarSessionNode.vue](packages/web/src/components/chat/SessionSidebarSessionNode.vue) - 会话节点
## 4. 方案概览
删除搜索框 + 提升颜色对比度 + 简化间距
## 5. 详细设计
### SessionSidebar.vue 修改
1. 删除搜索框区域（第270-282行）
2. 提升标题颜色：`text-muted-foreground/60` → `text-sidebar-foreground/80`
3. 提升分组标签颜色：`text-muted-foreground/50` → `text-muted-foreground/80`
4. 简化"最近访问"区域样式
### SessionSidebarSessionNode.vue 修改
1. 提升会话标题：`text-sidebar-foreground/70` → `text-sidebar-foreground`
2. 提升时间戳：`text-muted-foreground/40` → `text-muted-foreground/70`
3. 提升分支信息：`text-muted-foreground/50` → `text-muted-foreground/70`
## 6. 验收标准
- [ ] 搜索框已删除
- [ ] 所有文字对比度提升，肉眼清晰可见
- [ ] 明暗主题下都正常显示
- [ ] 布局更简洁，无多余嵌套感
