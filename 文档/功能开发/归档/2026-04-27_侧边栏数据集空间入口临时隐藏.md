# 侧边栏数据集空间入口临时隐藏

## 目标

- 暂时隐藏左侧侧边栏中的「数据集」和「空间」入口。
- 保留 `/datasets` 与 `/spaces` 路由和页面文件，避免删除尚未要求移除的能力。

## 实施

- 从 `packages/web/src/composables/useWorkbenchPrimaryNavigation.ts` 的 `workbenchPrimaryNavItems` 删除 `datasets` 与 `spaces` 两个展示项。
- 同步更新导航 composable 与 WorkbenchSidebar 单测中的导航项预期。
- 更新模块梳理与记忆文档，明确这是侧边栏展示层隐藏，不是路由删除。

- `npm run lint`
- `npm run typecheck`
