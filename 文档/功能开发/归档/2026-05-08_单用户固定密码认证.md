# 2026-05-08 单用户固定密码认证

## 目标

为 VPS 个人部署增加最小可用安全边界：固定密码登录、服务端 Session、全 API 鉴权、终端 WebSocket 鉴权。

## 范围

- 新增认证 spec：`文档/spec/单用户固定密码认证.spec.md`。
- 服务端新增认证运行时、登录/登出/会话接口、登录限流和全局 API 鉴权。
- 前端新增登录页、认证状态、路由守卫和 `401` 处理。
- 更新模块梳理中的认证边界。

## 非范围

- 不做多用户。
- 不做注册、找回密码、页面改密。
- 不做持久化 Session。

## 验收

- 服务端认证测试通过。
- 前端认证测试通过。
- 根目录 `npm run check` 通过。

## 完成记录

- 已实现固定密码登录、内存 Session、Cookie 鉴权、登录限流、全局 API 鉴权和终端 WebSocket upgrade 鉴权。
- 已实现 `/login` 页面、路由守卫、认证状态和 `401` 统一跳转。
- 已补充服务端与前端认证测试。
