# 2026-04-07 重构记录：极简现代化布局与主题修复

## 任务背景
用户要求彻底重构设计，删除顶部的两排冗余 UI，实现左侧可收缩菜单，并修复明亮模式切换不生效的问题。要求创建一个简约现代化的样式，无视现有陈旧样式。

## 核心变更内容

### 1. 架构与布局重构 ([packages/web/src/layouts/PlatformShell.vue](packages/web/src/layouts/PlatformShell.vue))
- **删除冗余顶部栏**：移除了原有的两排大标题、Badge 和路由按钮。
- **引入左侧导航栏**：
    - 实现了经典的侧边栏导航模式，包含：工作台、系统设置、主题实验室。
    - 支持 **收缩/展开** 功能（Collapse），收缩时仅显示图标，展开时显示文字标签。
    - 导航项根据当前路由自动高亮。
- **现代化主内容区**：
    - 主内容区背景设为微弱的混合色，增加层次感。
    - 内容区在 header（显示页面标题）下方流式排列，且具备最大宽度限制（1600px）以保证在大屏下的易读性。

### 2. 主题系统修复 ([packages/web/src/layouts/PlatformShell.vue](packages/web/src/layouts/PlatformShell.vue))
- **自动应用偏好**：在 `PlatformShell.vue` 的 `onMounted` 生命周期中显式调用 `applyCurrentPreference`。
- **修复明亮模式**：确保切换时 `document.documentElement` 的 `dark` 类名和 `color-scheme` 正确同步。
- **样式规范化**：移除了硬编码的颜色（如 `#09090b`），全面切换为使用 shadcn 定义的 CSS 变量（`bg-background`, `text-foreground`, `bg-card` 等），确保明暗模式下色彩都能自动适配。

### 3. 工作台组件样式对齐
- **对话列表重构 ([packages/web/src/components/chat/SessionSidebar.vue](packages/web/src/components/chat/SessionSidebar.vue))**：
    - 去掉了厚重的背景色和圆角，切换为与侧边栏一致的极简卡片风格。
    - 分类标题（最近使用、项目分组）改用更轻量的全大写字体和低对比度文字。
    - 会话项取消了渐变边框，改为悬浮变色，选中状态使用主题色（Primary）描边。
- **文件树重构 ([packages/web/src/components/workbench/ProjectFilePanel.vue](packages/web/src/components/workbench/ProjectFilePanel.vue))**：
    - 移除了琥珀色高亮，采用全系统统一的主题色前缀。
- **会话节点重塑 ([packages/web/src/components/chat/SessionSidebarSessionNode.vue](packages/web/src/components/chat/SessionSidebarSessionNode.vue))**：
    - 移除了原有的复杂操作按钮组，改为只有在 **Hover** 时才在右侧浮现的工具栏（重命名、固定、删除）。
    - 优化了选中状态的侧边指示条。
