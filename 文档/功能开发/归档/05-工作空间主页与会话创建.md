# 05 工作空间主页与会话创建

## 目标

实现工作空间主页输入框，并在首次发送后创建工作空间会话。

## 范围

- 主页标签。
- 统一输入框。
- 文本输入。
- 会话附件上传入口。
- 首次发送创建 Pi 会话和 ridge 会话索引。
- 模型、思考强度、Agent 临时选择。

## 不做

- 不做本机项目会话。
- 不做会话内任务面板。
- 不把临时切换写回全局设置。

## 验收

- 主页不是固定会话。
- 首次发送后才创建会话索引。
- 新会话加入左侧工作空间会话列表。
- 临时模型选择只影响后续消息。

## Spec 提取点

- 主页状态机。
- 首次发送 API。
- 会话索引创建字段。

## Spec 草案

### 状态

- empty_home
- sending_first_message
- conversation_created
- failed

### 行为

- 主页初始不绑定 Pi 会话。
- 首次发送创建 Pi 会话和 `session_index`。
- 创建成功后主页标签转换为会话标签。
- 临时模型、思考强度、Agent 只写入当前会话请求。

### 输出

- 新会话 ID。
- 会话索引记录。
- 首条消息流。

### 测试

- 打开主页不创建会话。
- 首次发送创建一条会话索引。
- 失败时主页保留输入。
- 临时模型不改全局设置。
