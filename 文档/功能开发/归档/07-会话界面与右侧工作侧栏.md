# 07 会话界面与右侧工作侧栏

## 目标

实现会话主界面和右侧工作侧栏。

## 范围

- 消息流展示。
- 复制消息。
- 编辑旧用户消息。
- 重试 AI 回复。
- 分叉新会话规则。
- 任务会话禁止分叉。
- 右侧侧栏：Summary、File tree、Git、Diff。

## 不做

- 不在会话内显示任务面板。
- 不删除消息。
- 不把工具过程做成独立页面。

## 验收

- 普通会话编辑/重试创建分叉。
- 任务处理会话不允许分叉。
- 工作空间和内部项目使用隐藏版本 diff。
- 外部 Git 项目使用项目 Git diff。

## 关联设计

- `文档/项目设计/会话界面与右侧工作侧栏.md`
- `文档/项目设计/工作空间隐藏版本管理.md`

## Spec 提取点

- 会话操作规则。
- 右侧侧栏 tab 数据来源。
- 分叉行为。

## Spec 草案

### 消息操作

- copy
- edit_user_message
- retry_assistant_message

### 行为

- 普通会话编辑旧用户消息创建分叉会话。
- 普通会话重试 AI 回复创建分叉会话。
- 任务处理会话禁用编辑分叉和重试分叉。
- 右侧侧栏按运行位置展示文件树和 diff。

### 测试

- 普通会话编辑旧消息生成新 session_index。
- 任务会话编辑旧消息入口不可用。
- 内部项目 diff 使用隐藏版本管理。
- 外部 Git 项目 diff 使用项目 Git。
