# 74 闪念展开详情与折叠

## 背景

闪念卡片展开后只展示原始内容、附件和状态，已处理闪念的 Agent summary 没有进入详情区。展开态也只保留左侧时间线圆点作为收起入口，用户很难发现如何再次折叠。

## 实现

- `packages/web/src/components/workspace/InboxView.vue`
  - 展开态增加顶部状态行和显式“收起闪念详情”按钮。
  - 已处理或 `analysisStatus='processed'` 的闪念展示 `recommendationText` 作为“处理结果”。
  - 原始内容独立成“原始内容”区块，附件和失败重试继续保留。
  - 折叠态增加稳定测试钩子 `data-testid="fleeting-note-collapsed"`。
- `packages/web/src/components/workspace/__tests__/InboxView.test.ts`
  - 覆盖已处理闪念展开后展示处理结果。
  - 覆盖展开后点击“收起闪念详情”回到折叠预览。

## 验收

- `pnpm --dir packages/web exec vitest run src/components/workspace/__tests__/InboxView.test.ts`
- `npm run check`
