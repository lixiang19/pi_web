# 62 侧边栏移除搜索与文件入口

## 目标

主左侧固定入口删除「搜索」和「文件」，保留工作台现有扁平侧栏结构。

## 实现

- `WorkspacePage.vue` 的 `fixedEntries` 不再包含 `search`、`files`。
- `WorkspaceSearchView.vue`、`FilesView.vue` 与对应 API 能力保留，不作为本次入口删除的目标。
- 模块梳理同步更新固定入口契约。

## 验收

- `WorkspacePage.test.ts` 固定入口顺序断言更新为：闪念、通知、任务、空间、终端、自动化、设置。
- 已先运行目标测试确认旧实现失败，再修改实现。
