# 笔记页面 Obsidian 风格重设计

**日期**：2026-04-28
**状态**：设计中

---

## 用户故事

> 用户打开笔记页面，希望看到一个**极简、专注、像 Obsidian 一样的写作环境**。左侧是干净的文件列表导航，中间是全宽的 Markdown 编辑器，可以同时打开多篇笔记并通过顶部标签页切换。所有内容自动保存，不需要手动点击保存按钮。右键菜单提供重命名、删除等文件操作。

**核心场景**：

1. 搜索并打开一篇笔记 → 全宽编辑 → 自动保存
2. 打开多篇笔记 → 标签页切换 → 每篇独立编辑状态
3. 新建笔记 → 弹出输入框 → 立即进入编辑
4. 右键菜单 → 重命名 / 删除笔记

│  ···菜单   │                                         │

### 与当前设计的差异

| 当前                                                 | 重设计后                                    |
| ---------------------------------------------------- | ------------------------------------------- |
| 侧栏 304px，带 "VAULT" 大写标签 + badge              | 侧栏 240px，极简无装饰标签，一个 header 行  |
| 编辑器 `max-w-[920px] border-x bg-card/65 shadow-sm` | 全宽无边距，`padding` 控制阅读区            |
| 编辑器 header 带 "EDITOR" 标签 + 大标题 + 保存按钮   | 无 header，标签栏承载文件名，保存状态在底部 |

| 手动保存按钮                                         | 自动保存（2s debounce）                     |
| 无标签页，同时只能看一篇                             | 多标签页，支持打开多篇                      |

## 交互逻辑

### 标签页系统

```
打开笔记 → 检查是否已在 tabs 中
  ├─ 已在 → 切换到该标签
  └─ 不在 → 创建新标签 → 加载内容 → 激活标签

切换标签 → 如果当前标签有未保存内容 → 先自动保存 → 切换到新标签

关闭标签 → 如果当前标签有未保存内容 → 先自动保存 → 移除标签
  ├─ 还有其他标签 → 激活相邻标签
  └─ 没有标签 → 空态

自动保存 → 编辑后 2s debounce → PUT /api/notes/content
  ├─ 成功 → 更新 savedContent → 状态变为"已保存"
  ├─ 失败 → 保持 dirty → 状态变为"保存失败"
  └─ 编辑中 → 状态变为"未保存"
```

### 文件操作（三点下拉菜单）

```
点击 ··· → DropdownMenu 弹出
  ├─ 重命名 → 内联编辑 / 弹窗输入新名称
  ├─ 删除 → 确认弹窗 → DELETE /api/notes → 刷新列表
  └─ 复制路径 → 复制 relativePath 到剪贴板
```

### 新建笔记

```
点击新建按钮 → 弹出轻量输入框（或直接创建"未命名.md"）
  ├─ 输入名称 → 回车确认 → POST /api/notes → 刷新列表 → 进入编辑
  └─ Esc 取消
```

interface NoteTab {
  relativePath: string;
  name: string;
  content: string; // 当前编辑器内容（可能 dirty）
  savedContent: string; // 最后一次成功保存的内容
  saveStatus: "saved" | "unsaved" | "saving" | "error";

## 视觉规范

### 侧边栏

- 宽度：240px
- 背景：`bg-sidebar` 主题变量
- 文字：`text-sidebar-foreground`
- 选中态：`bg-sidebar-accent text-sidebar-accent-foreground`
- 分割线：`border-sidebar-border`
- 无上部装饰性标签
- 文件列表只显示文件名
- 子目录通过缩进 padding-left 区分

### 标签栏

- 高度：36px
- 背景：`bg-muted/30`
- 激活标签：`bg-background` + 底部 2px primary 色指示线
- 非激活标签：`text-muted-foreground`
- 关闭按钮：hover 显示 × 图标

### 编辑器

- 全宽无边距：填满可用宽度
- 内边距：`px-8 py-6`（桌面）→ `px-4 py-4`（移动端）
- 字体：15px，行高 1.78
- 背景：`bg-background`
- 不需要 card 包裹、不需要 border

### 状态栏

- 高度：28px
- 背景：`bg-muted/20`
- 文字：`text-[11px] text-muted-foreground`
